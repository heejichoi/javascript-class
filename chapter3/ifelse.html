if ~ else

단순 if는 조건식이 true 일때만 시행되지만, if ~ else는 true일때와 false일때 실행하는 구문이 각각 존재함. 

            if(조건식1) {
               조건식1이 true면 실행될코드1; 
            } else if(조건식2){
                조건식2가 true면 실행될 코드2;
            }
            eles if (조건식 1,2,3,4,5...)
